## 🚀 사용 예시

```bash
python scripts/train.py \

    --models kobart \
    --use_wandb
```

### K-Fold 교차 검증
```bash
python scripts/train.py \
    --mode kfold \
    --models kobart \
    --k_folds 5
```

### 다중 모델 앙상블
```bash
python scripts/train.py \
    --mode multi_model \
    --models kobart llama-3.2-3b qwen \
    --ensemble_strategy weighted_avg
```

### Optuna 하이퍼파라미터 최적화
```bash

    --mode optuna \
```

