1. 문장형이 아닌 구나/절 형태로 작성
2. 중요 단어는 대문자로 시작 (Importance of Capitalize)
3. prefix 사용 예시:
   - feat: 기능 개발
   - fix: 오류 수정
   - docs: 문서 작업
   - test: 테스트 코드
   - conf: 환경설정
   - build: 빌드 관련
   - ci: CI 관련
   - chore: 기타 작업
   - style: 코드 포맷팅
4. prefix 뒤 작성되는 커멧 제목과 커밋 메세지 내용(설명)은 한글로 작성
   - 예시
      - feat: 다중 모델 앙상블 옵션 추가
         - 기능 상세 설명
